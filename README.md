# 🔮 JavaScript Fortune Teller

A simple, beginner-friendly JavaScript script that generates a random fortune using math and conditional logic. I built this project to practice the fundamentals of JavaScript control flow and random number generation.

## 🚀 Features Core Concepts Learned

* **Variables & Constants:** Using `const` for fixed data and `let` for values that change.
* **Math Object:** Utilizing `Math.random()` and `Math.floor()` to generate bounded random integers.
* **Conditional Logic:** Implementing `if / else if / else` structures to map numbers to specific outcomes.

## 🛠️ How It Works

1. The script defines 5 distinct fortunes.
2. It calculates a random integer between a `min` (1) and `max` (5) value.
3. An `if/else` chain checks the generated number and assigns the corresponding fortune to `selectedFortune`.
4. The final result is logged to the console.
